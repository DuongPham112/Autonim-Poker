# Deck Assets Folder

Place your card deck images here.

## Required Files
- 52 card images (e.g., `ace_spades.png`, `2_hearts.png`, etc.)
- `back.png` - The card back design

## Naming Convention
Cards should be named as: `{rank}_{suit}.png`

### Ranks:
- ace, 2, 3, 4, 5, 6, 7, 8, 9, 10, jack, queen, king

### Suits:
- spades, hearts, diamonds, clubs

### Examples:
- `ace_spades.png`
- `king_hearts.png`
- `10_diamonds.png`
- `2_clubs.png`

## Default Deck
The `default` folder should contain a complete 52-card deck.
You can add additional deck themes in separate folders.
